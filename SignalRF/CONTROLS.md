# Signal RF Quick Reference

## Main Menu
| Key | Action |
|-----|--------|
| `1` | WiFi Scanner |
| `2` | BLE Scanner |
| `3` | Ultrasonic Detector |
| `4` | Signal Mapper |
| `5` | Tracker Detector |

---

## WiFi Scanner
| Key | Action |
|-----|--------|
| `W` | Navigate up |
| `S` | Navigate down |
| `Enter` | View device details |
| `R` | Rescan networks |
| `Backspace` | Return to menu |

---

## BLE Scanner
| Key | Action |
|-----|--------|
| `W` | Navigate up |
| `S` | Navigate down |
| `Enter` | View device details |
| `R` | Rescan devices |
| `Backspace` | Return to menu |

---

## Device Detail View
| Key | Action |
|-----|--------|
| `R` | Scan and add point to graph |
| `Backspace` | Return to list |

---

## Ultrasonic Detector
| Key | Action |
|-----|--------|
| `T` | Play 1kHz test tone |
| `Backspace` | Return to menu |

---

## Signal Mapper
| Key | Action |
|-----|--------|
| `W` | Move cursor up |
| `A` | Move cursor left |
| `S` | Move cursor down |
| `D` | Move cursor right |
| `Space` / `Enter` | Sample position |
| `Q` | Toggle WiFi/BLE |
| `L` | View device list |
| `Backspace` | Return to menu |

---

## Tracker Detector
| Key | Action |
|-----|--------|
| `R` | Rescan |
| `L` | Toggle trackers/all devices |
| `W` | Scroll up |
| `S` | Scroll down |
| `H` | Help screen |
| `Backspace` | Return to menu |

---

## Status Bar Indicators
| Indicator | Meaning |
|-----------|---------|
| `SD` (green) | SD card detected |
| `XX%` | Battery level |
